მოწყობილობები

როგორ დავაყენოთ ან შევცვალოთ პაროლი Windows 10-ზე

  • საკითხავი დრო: 3 წუთი
  • წყარო: The SecDev Foundation
  • ბოლო განახლება: 08 თებერვალი 2022
  • კატეგორია: როგორ

ალბათ არ გინდათ, რომ ვინმემ დაათვალიეროს თქვენი ფაილები, დააინსტალიროს ჯაშუშური პროგრამები, ან შეცვალოს უსაფრთხოების პარამეტრები თქვენს კომპიუტერზე, როდესაც ახლომახლო არ ხართ? მაშინ მიჰყევით ქვემოთ მოცემულ ინსტრუქციებს, რათა დაიცვათ თქვენი Windows 10-ის მოწყობილობა პაროლით, ან შეცვალოთ უკვე არსებული.

 

გთხოვთ, გაითვალისწინოთ, რომ ამ ცვლილებების განსახორციელებლად, უნდა შეხვიდეთ თქვენს კომპიუტერში ადმინისტრატორის ანგარიშით.

დააყენეთ პაროლი მოწყობილობაზე

თუ Windows 10-ზე პაროლი ჯერ არ გიყენიათ, მიჰყევით ამ ნაბიჯებს. 

1. დააწკაპუნეთ ღილაკზე სტარტი თქვენი ეკრანის ქვედა მარცხენა მხარეს და გადადით განყოფილებაში პარამეტრები.

2. გადადით განყოფილებაში ანგარიშები. 

3. მენიუში მარცხნივ აირჩიეთ შესვლის ვარიანტები

4. აირჩიეთ პაროლი თქვენთვის ხელმისაწვდომი შესვლის ვარიანტების სიიდან და დააწკაპუნეთ ღილაკზე დამატება.

5. შეიყვანეთ პაროლი ველში, დაადასტურეთ და დააწკაპუნეთ ღილაკზე შემდეგი. დარწმუნდით, რომ გამოიყენეთ ძლიერი პაროლი.

6. დააწკაპუნეთ ღილაკზე დასრულება. აგრეთვე დარწმუნდით იმაში, რომ არ დაგავიწყდათ და გახადეთ პაროლის შეყვანა აუცილებელი პირობა თქვენს მოწყობილობაში შესასვლელად ყველასათვის, ვინც თქვენთან ერთად სარგებლობს თქვენი კომპიუტერით.

შეცვალეთ თქვენი მოწყობილობის პაროლი

მიჰყევით ამ ინსტრუქციებს უკვე არსებული პაროლის შესაცვლელად – მაგალითად, თუკი გსურთ პაროლის გაძლიერება. 

 

1. გადადით შესვლის პარამეტრების მენიუში (სტარტი > პარამეტრები > ანგარიშები > სისტემაში შესასვლელი პარამეტრები).

2. აირჩიეთ პაროლი თქვენთვის ხელმისაწვდომი სისტემაში შესასვლელი ვარიანტების სიიდან და დააწკაპუნეთ ღილაკზე შეცვლა.

3. შეიყვანეთ თქვენი მიმდინარე პაროლი და დააწკაპუნეთ ღილაკზე შემდეგი.

4. შეიყვანეთ ახალი პაროლი ველში, დაადასტურეთ ის და დააწკაპუნეთ ღილაკზე შემდეგი. დარწმუნდით იმაში, რომ იყენებთ ძლიერ პაროლს.

5. დააწკაპუნეთ ღილაკზე დასრულება.

გახადეთ მოწყობილობის პაროლის შეყვანა სავალდებულო

ახლა, როდესაც დააყენეთ მოწყობილობის პაროლი თქვენს Windows 10-ზე, თქვენ აგრეთვე უნდა გახადოთ პაროლის შეყვანა სავალდებულო ყველასთვის, ვინც ეცდება მიიღოს წვდომა თქვენს კომპიუტერზე. 

1. დააწკაპუნეთ ღილაკზე სტარტი თქვენი ეკრანის ქვედა მარცხენა კუთხეში და საძიებო ველში მოძებნეთ “netplwiz”. მთავარი შედეგი უნდა იყოს ამავე სახელწოდების პროგრამა. დააჭირეთ პროგრამას netplwiz მის გასახსნელად.

2. ფანჯარაში, რომელიც გაიხსნება, მონიშნეთ ველი – “ამ კომპიუტერის გამოსაყენებლად, მომხმარებლებმა უნდა შეიყვანონ სახელი და პაროლი”.

3. დააჭირეთ ღილაკს გამოყენება.

4. ცვლილებები რომ ამოქმედდეს, პაროგრამა მოგთხოვთ თქვენი პაროლის შეყვანას. შემდეგ დააწკაპუნეთ ღილაკზე OK.
