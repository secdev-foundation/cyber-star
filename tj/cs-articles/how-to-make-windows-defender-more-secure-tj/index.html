Чӣ тавр Windows Defender-ро амнтар метавон кард

  • Reading Time: 3 дақ
  • Сарчашма: The SecDev Foundation
  • Навсозии охирин: 01 August 2023
  • Навъ: Дастури амалӣ

ҲУШДОР: Ин мундариҷа барои корбарони дорои малакаи миёна то пешрафтаи истифодаи Windows таҳия шудааст. Иҷрои ин дастурҳо эҳтимол ба кори баъзе нармафзорҳои компютери шумо таъсири манфӣ расонад. Агар шумо аз чунин натиҷа роҳат нестед ва ё аз малакаи техникии худ шубҳа доред, тавсия медиҳем, ки ин дастурро татбиқ накунед.

Windows Defender

Windows Defender (инчунин маъруф бо номи Microsoft Defender) дар тӯли солҳои зиёд бо бастаи Microsoft Windows таъмин карда мешавад. Аммо дар бастани нармафзори зараровар чандон хуб набуд. Бо вуҷуди ин, дар солҳои охир имкони беҳтар кардани Windows Defender бо фаъол кардани баъзе хусусиятҳое, ки бисёре аз корбарони муқаррарӣ аз онҳо хабар надоранд, пайдо шуд. Бо чунин насбкуниҳо Windows Defender аз аксари антивирусҳои номдор ҳеҷ мондание надорад.
Матлаби мазкур барои фаъол кардани чунин хусусиятҳо роҳнамои шумо буда метавонад. Лутфан дар хотир дошта бошед, ки Windows ин хусусиятҳоро ба таври нобаёнӣ аз тарси он, ки ба мутобиқати барнома халал мерасонад, фаъол намекунад. Агар шумо ин хусусиятҳоро фаъол созед, баъзе барномаҳо метавонанд дуруст кор накунанд. Хушбахтона, барқарор кардани Windows Defender ба кори “муқаррарӣ” ҳамон қадар осон аст, ҳамчун бастани он. Дастурҳо дар бораи иҷрои ин амал низ дар ин ҷо оварда шудаанд.

Талаботи системаи амалиётӣ

  • Windows 10 Professional ё Enterprise версияи 1709 ё дертар. Шумо метавонед бо интихоби тугмаи Оғоз > Танзимот > Система > Дар Бораи > кадом версияи Windows-ро доред пайдо кунед. Дар доираи мушаххасоти Windows, шумо метавонед санҷед, кадом нашр ва версияи Windowsро дастгоҳи шумо истифода мебарад.
  • Агар шумо Windows 10 Home -ро истифода баред, ин дастурҳо ба шумо кӯмак намекунанд. Дар ин ҳолат, шумо бояд дастгоҳи худро тавассути харидани як нармафзори антивируси боэътимоди тарафи сеюм муҳофизат кунед.
  • Бисёре аз ин танзимот кор намекунанд, агар компютери Windows-и шумо қисми домен бошад ва мудири системаи шумо танзимоти алтернативӣ гузошта бошад.

Таъмини Microsoft Defender

  1. Configure Defender-ро зеркашӣ кунед ё рамзи сарчашмаи онро дар ин ҷо тафтиш кунед. Ин утилит танҳо интерфейси графикии корбар (GUI) аст, ки муҳофизати Windows Defender-ро осонтар мекунад.
  2. Пас аз зеркашӣ, асбобро ду маротиба клик кунед. Шумо метавонед ҳар гуна танзимоти амниятиро, ки ба шумо маъқул аст, интихоб кунед, аммо барои муҳофизати ҳадди аксар тавсия медиҳем, ки тугмаи High-ро пахш кунед ва пас аз он имконоти мувофиқро ба скриншотҳои дар поён оварда шуда тағйир диҳед. Ин танзимот тавозуни хуби умумии байни амният ва нороҳатиро, ки танзимот метавонад ба шумо расонад, таъмин мекунад.
    • Танзимоти интихобшуда (берун аз High) инҳоянд: Дуздии маълумоти эътимодномаро аз зерсистемаи мақомоти маҳаллии амниятии Windows манъ кунед (lsass.exe) – ON
    • Офаридаҳои равандро, ки аз фармонҳои PSExec ва WMI сарчашма мегиранд, блок кунед – ON
    • Дастрасӣ ба ҷузвдонҳои назоратшаванда – ON


  3. Пас аз анҷом додани ин тағйирот, Close-ро клик кунед ва компютери худро бозоғоз намоед.

Бартараф кардани танзимоти антивируси Windows Defender

Агар пас аз ворид кардани тағйиротҳои дар боло тавсифшуда шумо баъзе барномаҳоро кушода натавонед, шумо ҳамеша метавонед Windows Defender-ро ба танзимоти аслӣ баргардонед ва асбоби Configure Defender -ро боз кушоед ва тугмаи “Пешфарз” ва сипас Пӯшед. Шумо бояд компютери худро бозоғоз намоед, то ин ки тағйирот эътибор пайдо кунад.

  • Windows Defender бо танзимоти пешфарз
